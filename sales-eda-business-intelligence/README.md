# Sales Data EDA & Business Intelligence

This project performs Exploratory Data Analysis (EDA) on a sales dataset.

## Dataset
The dataset contains:
- Order ID
- Customer Name
- Region
- Category
- Sales
- Quantity
- Discount

## Analysis Performed
- Data exploration
- Descriptive statistics
- Sales visualization
- SQL business queries
- Correlation analysis

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- SQLite
- Google Colab
