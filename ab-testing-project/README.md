# A/B Testing Data Storytelling Project

## Overview
This project performs A/B testing to evaluate whether a new website layout improves user conversion rates. The analysis includes data exploration, visualization, and statistical hypothesis testing.

## Dataset
File: `ab_testing_dataset.csv`

Columns:
- user_id : Unique user identifier
- group : Control or Treatment group
- converted : Conversion result (0 = No, 1 = Yes)

## Project Workflow
1. Load and explore dataset
2. Perform data visualization
3. Calculate conversion rates
4. Conduct hypothesis testing using T-test
5. Interpret results and generate business insights

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
- Statsmodels
- Google Colab / Jupyter Notebook

## Files in Repository
- `ab_testing_analysis.ipynb` : Complete analysis code with outputs
- `ab_testing_dataset.csv` : Dataset used for analysis
- `presentation.pptx` : Project presentation slides
- `requirements.txt` : Python dependencies

## Conclusion
The project demonstrates how A/B testing and statistical analysis can help businesses make data-driven decisions to improve website conversion rates.
